Cómo Trabajo

Soy desarrollador web trabajando con arquitecturas modernas e integración de IA. Básicamente combino lo mejor de la inteligencia artificial actual con infraestructura cloud sólida.

Las IAs que uso

  • Claude Desktop — Lo uso para planificar proyectos, escribir documentación y resolver problemas técnicos complicados
  • Claude Code — Directo en VSCode con acceso a la terminal. Desarrollo asistido en tiempo real
  • Google Antigravity (con Gemini) — IDE agéntico ideal para prototipar interfaces rápido
  • Fish Audio — Para generar voz ultra-realista. Perfecto para audiolibros y contenido en +70 idiomas

Hosting e infraestructura

  • Hostinger — Mi hosting principal. SSH + rsync para deployments limpios
  • Vultr — VPS para los backends Python con FastAPI
  • Cloudflare — DNS, CDN, Pages y Workers. El paquete completo para rendimiento global

Stack técnico

Frontend:

HTML/CSS/JS, SASS, React cuando es necesario

Backend:

Python con FastAPI + Uvicorn, Node.js

Herramientas:

Puppeteer para PDFs, Wrangler CLI, Git/GitHub
